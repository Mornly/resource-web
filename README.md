# 📦 Resource Share Worker
基于 Cloudflare Workers 构建的轻量级资源分享程序

**纯 Serverless 架构 · 多存储方案支持 · 自带安全防护 · 苹果风精致界面**

[功能特性](#-功能特性) · [部署教程](#-部署教程) · [使用指南](#️-使用指南) · [安全说明](#-安全说明)

## ✨ 功能特性

### 🎨 前端界面
- **苹果风设计**：毛玻璃质感、圆角卡片、顺滑过渡动画，还原原生系统视觉体验
- **明暗双主题**：一键切换浅色/深色模式，本地记忆用户偏好，下次访问自动生效
- **响应式布局**：完美适配桌面端、平板、手机，全设备一致体验
- **交互动效**：卡片悬浮上浮、模态框淡入缩放、平滑滚动，细节拉满
- **资源展示**：瀑布式卡片布局、自动分页、关键词搜索，浏览高效

### 📁 多存储方案
- **Telegram Bot 存储**：通过 Telegram 机器人中转文件，免费无容量限制
- **Cloudflare R2 存储**：内置 R2 支持，预览图和资源文件均可存入
- **URL 外链模式**：直接填写外部文件链接，无需占用自身存储
- **原始文件名保留**：下载时完整保留上传时的文件名与后缀，无信息丢失

### 🔐 安全防护
- **CSP 内容安全策略**：内置安全响应头，有效防御 XSS 注入、数据劫持等攻击
- **XSS 全链路转义**：前后端双重 HTML 转义，所有用户输入均做安全过滤
- **登录暴力破解防护**：同一 IP 连续 5 次登录失败，自动锁定 15 分钟
- **文件类型白名单**：严格限制可上传的图片与文件格式，禁止恶意文件上传
- **下载限流机制**：按 IP 限制每日下载次数，可自定义，防止站点被滥用

### 🛠️ 管理功能
- **独立管理后台**：管理员账号密码登录，Basic Auth 鉴权
- **资源全生命周期管理**：支持资源的新增、编辑、删除、查询
- **访问问答保护**：可为单个资源设置验证问题，答对才可下载
- **站点全局设置**：自定义网站标题、下载限流阈值、一键开启维护模式
- **标签系统**：支持多标签分类，配合搜索快速定位资源

## 📦 部署教程

### 前置准备
1. 一个 **Cloudflare 账号**（免费套餐即可使用）
2. Telegram 账号，用于创建 Bot 存储文件
3. （可选）自定义域名，可绑定到 Workers 服务

---

### 快速部署（复制粘贴，30秒完成）
1. 打开 Cloudflare 后台，进入 **Workers & Pages** → **创建应用程序** → **创建 Worker**
2. 自定义 Worker 名称，点击「部署」
3. 进入 Worker 详情页，点击「编辑代码」
4. 把 `index.js` 的全部代码复制粘贴进去，点击「部署」

---

### 必选配置：KV 命名空间绑定
所有数据存储依赖 KV，必须配置：
1. 进入你的 Worker → **设置** → **变量**
2. 找到 **KV 命名空间绑定** → **添加绑定**
   - 变量名（必须严格一致）：`KV`
   - 点击「创建新命名空间」，自定义名称后选中
3. 点击「保存并部署」生效

---

### Telegram Bot 存储
1. Telegram 搜索 **@BotFather**，发送 `/newbot` 按指引创建机器人，获取 **Bot Token**
2. 创建一个公开频道/群组，将机器人设为管理员
3. 获取频道/群组 ID（可通过 @userinfobot 等工具获取，格式通常为 `-100xxxxxx`）
4. 回到 Worker 变量设置，添加两个**环境变量**：
   - `TG_BOT_TOKEN`：你的机器人 Token
   - `TG_CHANNEL_ID`：你的频道/群组 ID
5. 保存并重新部署生效

---

### R2 存储桶绑定
需要本地上传预览图、使用 R2 存文件时配置：
1. Cloudflare 后台进入 **R2** → 创建存储桶（名称自定义）
2. 回到 Worker → **设置** → **变量** → **R2 存储桶绑定** → **添加绑定**
   - 变量名（必须严格一致）：`R2_PREVIEWS`
   - 选择你创建的存储桶
3. 保存并重新部署，解锁本地上传功能

---

### 管理员账号配置
在 Worker 变量设置中添加两个**环境变量**：
- `ADMIN_USERNAME`：管理员登录用户名
- `ADMIN_PASSWORD`：管理员登录密码

> ⚠️ 请设置强密码，避免被暴力破解

## 🛠️ 使用指南

### 进入管理后台
访问你的站点，在地址栏末尾加上 `/#admin` 回车即可进入登录页
示例：`https://你的域名/#admin`

### 添加资源
1. 登录后台 → 切换到「资源管理」→ 点击「添加资源」
2. 填写基础信息：资源名称、资源ID（唯一标识）、标签、描述（支持 Markdown）
3. 选择存放方式：
   - **本地上传**：选择存储位置（Telegram / R2），上传预览图和资源文件
   - **URL外链**：直接填写预览图和文件的外部链接地址
4. （可选）开启「启用访问问题」，设置问题和答案
5. 点击保存，资源立即生效

### 站点设置
切换到「基本设置」，可配置：
- 网站标题
- 单 IP 每日下载次数（1/5/10/50/100/无限）
- 网站关停（开启后前台仅显示维护页）

## 🔒 安全说明
1. 所有用户输入均经过 HTML 转义，杜绝 XSS 注入风险
2. 登录接口有频率限制，有效抵御暴力破解
3. 上传文件严格校验 MIME 类型，禁止可执行文件上传
4. 内置 CSP 安全策略，限制资源加载来源
5. 管理员密码仅存储在 Cloudflare 环境变量中，代码中不包含任何敏感信息

## ⚠️ 注意事项
1. **Telegram 文件限制**：Bot 单文件最大支持 50MB 上传，大文件建议使用 R2 或外链
2. **R2 计费说明**：Cloudflare R2 有免费额度，超出后按量计费，注意用量
3. **数据备份**：建议定期导出 KV 命名空间数据，防止误删
4. **维护模式**：开启后仅管理员后台可访问，前台全部显示维护页面
